1 OVERVIEW

Problem Space

Practical human diving depth is only 300 meters deep. However, a lot of work need to be done in the deeper sea: clean the attached sea creature of the underwater equipment, troubleshooting, cut the cable, salvage and etc. So, a deepwater robot arm is necessary.

3.2 Scenarios

The Robot should be installed on a deep-sea submersible craft. The submersible craft wanders in the bottom of the sea. Once it identifies a task, it will install appropriate tool on its wrist.

E.g. When the task is cleaning the microorganism. It will install a cleaning brush and clean the pipe.
